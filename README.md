## Solidus

> Simple store demo with Solidus, then deploy to Heroku by Github Action.

### Prerequisites

- Ruby version: 2.7.2
- Node version: 14.15.4
- Rails 6.1.1
- Solidus 3.0.0.rc1

### Development

* Run the Rails app

```sh
docker-compose -f docker-compose.dev.yml up -d
```
